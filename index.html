What is Lando?

Free and open source

Cross-platform

Local development environment and DevOps tool

Built on Docker container technology

  • Set up to mimic your production environment
  • Standardize your dev environments
  • Integrate with Pantheon
  • Store configuration in .yml file
  • Works with D6, D7, D8, Backdrop, WordPress, Joomla and Laravel
  • Also Python, Go, Ruby, Dotnet and barebones LAMP stack
  • Add Redis, Solr or Varnish
